# Antd-Admin-Template

github 地址: [AntdAdminTemplate](https://github.com/mrleidesen/antd-admin-template)

已经集成如下：

- vite3
- windi css
- eslint
- prettier
- react-query v4
- vitest (根据情况更新，版本比较低)
- valtio
- react-router-dom
- axios
- husky
- lint-stage
- antd v5

均锁版本，如果需要升级，请自行更新，模板也会不定期更新

## 提示

代码中一些需要根据实际情况修改的地方都标注注释 `FIXME:` ，可以全局搜索进行修改

## 运行

> pnpm install

> pnpm dev
